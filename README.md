# DeadTreeCore
如何使用D3力导向图构建知识图谱

------------------
> 注册账号：只能注册专家账号 登录使用注册邮箱
> 融合算法：目前的知识图谱融合算法采用完全相同融合法 需要通过二重循环判断重复节点
> 使用注意：由于知识图谱的构建采用了D3.JS提供的力导向图CDN 因此在使用本系统时需要联网
> 必要环境：MS.NET CORE 2.0运行支持 Chrome 53以上版本的浏览器（必须支持高级JS方法）
